Acerca de esta App

¡Hola! Soy Jaime, alumno del Instituto IES Ruiz Gijón, y esta Aplicación de Peliculas de Animación es una práctica completa creada en Angular, diseñada para aprender a consumir APIs y construir aplicaciones modernas.

La aplicación está organizada en una estructura clara y modular: en el componente principal (AppComponent) encontrarás las zonas de header, main y footer. Dentro del main se utiliza el <router-outlet> para cargar las distintas vistas según la navegación.

La navegación se realiza mediante un navbar en el header, desde donde puedes acceder a las diferentes secciones:

  • Bienvenida
  • Búsqueda de películas a través de la API propia
  • Acerca de esta aplicación
  • Página 404 para rutas no encontradas
  • Admin (Para el uso de la API Propia)

Entre las características técnicas implementadas destacan:

  • Uso de ngModel para enlazar el input de búsqueda con el modelo de datos.
  • Directivas estructurales *ngIf y *ngFor para mostrar condicionalmente el contenido y listas dinámicas.
  • Implementación de una pipe personalizada para transformar datos y mejorar la presentación.
  • Definición y uso de la interfaz IPelicula para tipar los datos rescatados desde la API propia.
  • Estilos responsivos y modernos con Bootstrap, garantizando una buena experiencia en dispositivos móviles y escritorio.

El proyecto contempla la integración semi-opcional del CRUD completo mediante el consumo de otra API, con componentes y servicios dedicados, ampliando así las funcionalidades.
